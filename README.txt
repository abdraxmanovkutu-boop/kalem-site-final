Сайт 'Издательский дом Калем'

Как открыть:
1. Распакуйте архив.
2. Откройте файл index.html в браузере.

Панель аналитики:
- На сайте нажмите 'Смотреть аналитику' или перейдите в раздел 'Аналитика'.
- Логин: admin
- Пароль: Kalem2026!

Что есть внутри:
- Главная страница
- Каталог книг
- Отдельный раздел 'Тетради-тренажёры'
- Блок 'О нас'
- Контакты и форма заявки
- Опрос по возрасту и региону
- Скрытая аналитика: посещения, просмотры, клики, избранное, заказы, возраст, регионы, топ товаров

Важно:
Данные сохраняются в localStorage браузера, то есть работают локально на одном устройстве.
